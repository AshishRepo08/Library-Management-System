Welcome To My Library Management Application. This application in its current version can serve as a library management system for a book store, or a education institute library.

**Tech Stack Used**:
 - Lanagugage: Java
 - Backend: Spring Boot, Spring MVC, Spring Security, JPA/Hibernate, REST API
 - Databases: PostgreSQL
 - Testing: JUnit, Mockito
